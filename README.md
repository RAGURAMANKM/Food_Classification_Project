# Food_Classification_Project
This project will helpful for diet planning customers

The Food Classification System is a Machine Learning project that predicts the food category based on its nutritional values and food-related attributes.We done data preprocessing, exploratory data analysis, feature engineering, dimensionality reduction, model training, evaluation, and prediction based on given data.The Food Classification System successfully predicts food categories from nutritional information using supervised machine learning techniques. The project follows a standard machine learning workflow by preprocessing the data, splitting it into training and testing sets before scaling and PCA, evaluating multiple classification models, selecting the best-performing model, and generating accurate predictions for new food records. This project demonstrates practical skills in data preprocessing, feature engineering, dimensionality reduction, model evaluation, and database integration using PostgreSQL.
